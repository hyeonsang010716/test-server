#### 3. 미드저니 활용법
- **설치 및 설정**
  - 구글에서 "미드저니" 검색 후 설치
  - 디스코드 설치 및 로그인
  - 디스코드에서 Midjourney bot 채널 생성 및 추가
- **구독정책**
  - Basic Plan: 200개의 이미지 생성
  - Standard Plan: 15시간 이미지 생성
  - Pro Plan: 30시간 이미지 생성 (스텔스 이미지 생성 가능)
  - Mega Plan: 60시간 이미지 생성 (스텔스 이미지 생성 가능)
- **이미지 생성**
  - /imagine 명령어 사용
  - 프롬프트 작성법: 단어 선택, 길게 서술하지 않기, 핵심 단어 사용
  - Multi prompts 활용: 단어들을 분리해 묘사, 가중치 적용
  - 이미지와 Text 조합, 이미지와 이미지 조합
  - Parameter 작성법: 비율(aspect), 제거(no), 스타일라이즈(Stylize)

#### 4. 실습 예제
- 앨범 커버
- 동화책 만들기 (헨젤과 그레텔, 잭과 콩나무, 토끼와 거북이)
- 스토리보더 활용

#### 5. 용어정리
- **Prompt**: 이미지를 생성하기 위해 필요한 문장
- **Inpaint**: 원하는 영역을 부분적으로 수정하는 방식
- **Outpaint**: 내용, 패턴 등을 분석하여 이미지의 바깥 부분을 자연스럽게 변화/응용하는 이미지 제작 방식
